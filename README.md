# Chrome Morph

> Скажи любому сайту, что с ним сделать.

Расширение для Chrome, которое меняет любую открытую страницу по твоей фразе на русском или английском. Поменять цвета, увеличить шрифты, спрятать баннеры, переделать в стиле «Рик и Морти» — одной строчкой.

[![Презентация](https://img.shields.io/badge/Презентация-✨-b86cff?style=for-the-badge)](https://morph.hugmediary.com/presentation)

## Как это работает

1. На любой странице жмёшь **⌘+J** (Mac) или **Ctrl+J** (Win/Linux)
2. В центре появляется поле ввода
3. Пишешь, что хочешь:
   - «сделай шрифт побольше»
   - «скрой все картинки»
   - «greyscale»
   - «удали все слова, начинающиеся на букву П»
   - «переделай в стиле Рик и Морти»
4. Или жмёшь один из 8 готовых пресетов
5. Сайт меняется. Если что-то не так — кнопка «отмена» возвращает как было

## Установка (через Developer mode)

Расширение ещё не в Chrome Web Store, ставится вручную:

1. **Скачай этот репозиторий**
   - Жми зелёную кнопку **Code → Download ZIP** наверху страницы
   - Распакуй архив куда удобно (например, в Documents)

2. **Открой страницу расширений в Chrome**
   - Введи в адресной строке: `chrome://extensions`
   - В правом верхнем углу включи **Developer mode** (Режим разработчика)

3. **Загрузи расширение**
   - Нажми **Load unpacked** (Загрузить распакованное)
   - Выбери папку, куда распаковал ZIP (там должен быть `manifest.json`)
   - Готово — иконка ✨ появится в панели расширений

4. **Закрепи иконку (по желанию)**
   - Жми на пазл в правом верхнем углу Chrome
   - Рядом с «Chrome Morph» жми кнопку-кнопку pin

5. **Открой любой сайт** и нажми **⌘+J** или иконку расширения

## Хоткей

По умолчанию **⌘+J** (Mac) / **Ctrl+J** (Win/Linux).

Если конфликтует с чем-то — перенастрой здесь: `chrome://extensions/shortcuts`

## Встроенные режимы

| Эмодзи | Режим | Что делает |
|--|--|--|
| 👵 | Для бабушки | Увеличивает все шрифты на 40%, межстрочный 1.5 |
| ⠃ | Брайль | Шрифт Noto Sans Symbols 2 для всего текста |
| ⚫ | Greyscale | Полностью серая палитра (CSS-фильтр) |
| 🪩 | Дискотека | Мерцание, hue-rotate, цветомузыка |
| 💻 | Хакер | Зелёный моноширинный на чёрном |
| 📖 | Читалка | Только текст, без баннеров и сайдбаров |
| 💥 | Мультяшно | Comic Sans, яркие цвета |
| 🛸 | Рик и Морти | Полный редизайн в стиле сериала |

## Своими словами

Просто опиши в поле ввода что хочешь. Расширение само определит:
- **стиль** — мелкие CSS-правки (цвета, шрифты, спрятать что-то)
- **контент** — JS-патч для изменения содержимого (замена текста, эмодзи вместо картинок)
- **редизайн** — большая перестройка (полная смена темы)

Или выбери конкретный режим кнопками **авто / стиль / контент / редизайн** в окне.

## Кнопки в окне

- **Применить** — отправить запрос (или жми Enter)
- **↶ Отмена** — отменить последнее изменение
- **↻ Reset** — сбросить всё и обновить страницу
- **Esc** — закрыть окно

Во время генерации окно сворачивается в правый нижний угол с крутящейся анимацией. Можно нажать на него чтобы развернуть обратно.

## Что под капотом

- Расширение шлёт фразу + содержимое страницы на сервер `morph.hugmediary.com`
- На сервере микро-классификатор выбирает категорию (стиль/контент/редизайн) и зовёт LLM
- В ответ приходит CSS+JS-патч, который применяется к странице
- Все запросы анонимные, привязаны к случайному UUID (создаётся при первом запуске)

Серверный код в отдельном приватном репозитории — здесь только клиентская часть.

## Privacy

- Когда применяешь морф, на сервер уходит: твоя фраза + HTML страницы (без `<script>`/`<style>`)
- Логируется: фраза, URL, домен, анонимный UUID, длительность запроса
- Никаких куков, никакого логина, никакой персональной информации

Если не хочешь чтобы что-то улетало на сервер — просто не пользуйся расширением на этом сайте.
